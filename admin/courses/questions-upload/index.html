문제은행 JSON 업로드

흐름도 기준 과정별 JSON을 검증·미리 보기 후 DB에 적재합니다. 로그인 후 절대 관리자 계정만 사용할 수 있습니다. 한 번 저장하면 해당 과정의 활성 문제은행 세트가 교체됩니다.

적용 과정 및 현황

  • 불러오는 중…

JSON 파일 선택 (최대 약 10MB)

파일을 선택하면 내용만 메모리에 올립니다. 저장 전까지 서버에 반영되지 않습니다.

검증/미리보기 응답

{
  "questions": [
    {
      "question": "문항 본문",
      "options": ["① 선택지", "② 선택지", "③ 선택지", "④ 선택지"],
      "answer": 2,
      "subject": "과목명",
      "explanation": "(선택) 해설 — 메이크업 JSON의 category/options 객체 형식도 지원합니다."
    }
  ]
}
