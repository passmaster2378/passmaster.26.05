학습 현황

승인된 과정별로 현재 단계·진행률·12단계 카드 상태를 확인합니다. 문서상 URL은 이 사이트의 정적 경로와 쿼리로 아래처럼 연결되어 있습니다.

학습 페이지 URL 매핑 (구조안 → PASSmaster)

동적 ID는 각 수강의 실제 신청 ID로 바꿉니다. 과정 허브는 my-courses/enrollment-001/index.html?id={enrollmentId} 형태로 부릅니다.

역할 구조안 예시 URL 사이트 경로
내 강의실 /my/courses /my-courses/
과정 학습 홈 /learn/:courseId /my-courses/enrollment-001/index.html?id={id}
단계 시작 안내 /learn/…/stage/:n/start /study/makeup/index.html?enrollmentId={id}&stage={n}&view=start
문제풀이 /learn/…/stage/:n/play /study/makeup/index.html?enrollmentId={id}&stage={n}&view=play
단계 결과 /learn/…/stage/:n/result /study/makeup/index.html?enrollmentId={id}&stage={n}&view=result
전체 통계 리포트 /learn/:id/final-report /study/makeup/index.html?enrollmentId={id}&view=final-report (리포트 UI는 과정별 연동 시 확장)
수강 완료 /learn/:id/complete /study/makeup/index.html?enrollmentId={id}&view=complete (완료 화면 과정별 연동)
